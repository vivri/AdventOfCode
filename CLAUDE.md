# Advent of Code 2025 - Day 12

## Usage

```bash
# Install requirements 
pip install -r requirements.txt

# Run on puzzle input (quiet mode - prints only the answer)
python aoc_12_2025.py aoc_12_2025_input.txt

# Run with verbose output (shows each region result and filled grids)
python aoc_12_2025.py aoc_12_2025_input.txt --verbose
```

## Code Style

- Type hints throughout
- Frozen dataclasses for immutability
- Static methods for helpers
- `_printv()` wrapper for conditional verbose output
